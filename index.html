Quick reproduction

Clone repo, create venv, install requirements, run demo.

git clone https://github.com/NaomieBambara/Text-to-Speech--HandSyllables.git
cd Text-to-Speech--HandSyllables
python -m venv venv
# Windows PowerShell
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
python tonal_tts_full.py --demo

Highlights

  • Hand-segmented syllable corpus (1–100 sentences) with tone labels (1 low / 2 mid / 3 high).
  • Orthography → token lexicon with diphthong/triphthong alternates for flexible synthesis.
  • Concatenative synthesizer with amplitude normalization, short inter-token silence, and optional pitch-preserving transforms.
  • Objective MFCC+DTW evaluation and draft MOS-style listening tests.
